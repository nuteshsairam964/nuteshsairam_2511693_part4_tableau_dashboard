# Executive Business Dashboard using Tableau

## Project Overview

This project presents an interactive executive dashboard developed using Tableau to analyze retail sales performance and support business decision-making. The dashboard combines sales, profit, customer, regional, shipping, discount, and return analysis into a single interactive interface.

The objective is to help business stakeholders monitor key performance indicators, identify profitable business opportunities, and make data-driven strategic decisions.

---

# Business Objectives

The dashboard aims to:

* Monitor overall business performance.
* Analyze sales trends over time.
* Compare regional sales and profitability.
* Evaluate product category performance.
* Understand customer segment behavior.
* Analyze the impact of discounts on profitability.
* Monitor shipping performance.
* Identify return patterns.
* Support interactive business exploration through filters and dashboard actions.

# Business Insights

Key insights obtained from the dashboard include:

* Sales show an overall positive trend.
* Technology products generate the highest profitability.
* Corporate and Home Office customers contribute significant revenue.
* Higher discounts reduce profitability.
* Standard Class is the most frequently used shipping mode.
* Regional performance varies significantly across states.
* Product returns negatively affect profitability.
